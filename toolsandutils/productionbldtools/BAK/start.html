Welcome to the Symbian OS Binary Access Kit
The OS Binary Access Kit is for companies making hardware and software products for Symbian OS. It enables the development of driver, middleware and prototype application software for Symbian OS. It can be used to extend and modify Symbian OS source code, and rebuild components to be supplied to Symbian or to Symbian OS licensees for integration into products. The OS Binary Access Kit contains:

  • documented application and system development APIs
  • build tools for emulators, target hardware and ROM building tools
  • the TechView UI for launching and controlling programs and enabling GUI-based testing.

Symbian OS developer products and support
The Symbian Developer Network provides a range of support for Symbian OS developers, from the Symbian OS knowledgebase to developer and white papers and the annual Symbian Smartphone Show. With support for C++, Java and OPL development, Symbian Developer Network offers developers the training and resources needed to understand and develop for Symbian OS phones. Registering with Symbian Developer Network gives developers access to:

  • free developer and white papers
  • Symbian OS knowledgebase
  • software downloads
  • technology newsgroups and newsletters
  • membership packages.
